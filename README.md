# URJA System Upgrade — Rocket Moon Launch Animation

A futuristic, high-fidelity, performance-optimized rocket launch animation built with pure HTML5, CSS3, and JavaScript (`requestAnimationFrame`).

## Features
- 🚀 **Smooth Dynamic Rocket**: Rising and continuously accelerating using a non-linear physics curve (`t^2.5`), growing in size dynamically as it goes up.
- 🔥 **4-Layer Engine Flame & Exhaust Particles**: Multi-layer flickering SVG flame exhaust paired with JavaScript-generated particle embers falling downwards.
- ⚡ **Audio Synthesis**: High-immersion, synchronized audio built on the Web Audio API (humming, engine charging pitch shifting, launch whoosh, landing chime).
- 🌕 **Centred Moon Backdrop**: A massive `1100px` lunar backdrop on the initiatives page with crater shadows, lit crescent limb shimmer, and twinkling starfields.
- 💻 **Initiatives Dashboard**: Transitioning into a clean, modern grid featuring 6 projects with smooth hover interactions.

## Deployment on Vercel

This is a zero-dependency, static website. To deploy it to Vercel:

1. Go to the [Vercel Dashboard](https://vercel.com).
2. Click **Add New** > **Project**.
3. Import this GitHub repository (`AadityaAggarwal2007/moon-animation`).
4. Vercel will automatically detect the static project. Click **Deploy**.
